# เงินไทย — แพลตฟอร์มบัญชี-ภาษี-วางแผนการเงิน

แพลตฟอร์มสำหรับบุคคลธรรมดาและผู้ประกอบการรายย่อยในไทย คำนวณภาษีอ้างอิงประมวลรัษฎากร ปีภาษี 2569

## ฟีเจอร์

- บันทึกบัญชี (ติ๊กประเภทเงินได้ ม.40)
- คำนวณภาษีเงินได้ + ลดหย่อนครบ 26 รายการ
- ภาษีครึ่งปี ภ.ง.ด.94 + จำลองแบบยื่น
- VAT ภ.พ.30
- ภาษีหัก ณ ที่จ่าย
- ต้นทุนสินค้า FIFO / Weighted Average
- วิเคราะห์รายเดือน-ปี + Export Excel/CSV/PDF
- วางแผนการเงิน

## วิธีรัน

streamlit run app.py
